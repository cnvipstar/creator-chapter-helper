# Chrome 应用商店发布检查清单

## 开发者账号

- 登录 Chrome Developer Dashboard。
- 注册 Chrome Web Store 开发者账号。
- 支付一次性注册费用。

## 商品资料

- 上传扩展 zip 包。
- 填写名称、简短说明和详细说明。
- 选择类别：生产工具。
- 上传 128x128 图标。
- 上传至少 1 张截图，建议上传 3 张。
- 上传 440x280 小推广图。
- 填写隐私政策 URL。
- 填写权限用途说明。
- 填写审核测试说明。
- 价格选择免费，不配置付费或订阅。

## 隐私

- 声明扩展不收集、不出售、不共享用户数据。
- 声明章节文本只在本地浏览器中临时处理。
- 声明不读取 Cookie、密码、浏览器历史或无关页面数据。
- 声明不会点击平台最终“发布”按钮。
- 声明错误日志只保存在本地、只保留当天、不会上传服务器。

## 发布前验证

```bash
npm test
npm run check
npm run package
```

## 人工验证

- 在 Chrome 中加载未打包扩展。
- 刷新抖音或小红书视频发布页。
- 粘贴章节文本。
- 执行写入。
- 确认章节写入成功。
- 确认最终“发布”按钮没有被点击。
