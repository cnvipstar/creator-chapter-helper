# Chrome 应用商店上架文案

## 名称

创作者章节助手

## 简短说明

把粘贴的章节时间轴批量写入抖音和小红书创作者平台。

## 详细说明

创作者章节助手帮助创作者把已经整理好的章节文本批量写入视频章节编辑器，减少重复拖进度条、输入章节名和简介的手工操作。

当前支持抖音创作者中心和小红书创作服务平台，适合课程、知识分享、教程、长视频切片等需要清晰章节结构的创作者。

主要功能：

- 粘贴 `00:05-00:28｜标题：简介` 形式的章节文本。
- 自动解析开始时间、结束时间、章节名和章节简介。
- 写入前预览章节列表。
- 按平台校验章节限制：抖音章节名 12 字、简介 100 字；小红书章节名 14 字。
- 支持自动截断超长章节名和简介。
- 可选择写入前清空已有章节。
- 在页面内显示执行进度。
- 写入失败时可复制当天本地错误日志，方便排查页面结构变化。
- 只保存章节弹窗，不会点击最终发布按钮。

隐私说明：

扩展不上传章节文本，不收集账号信息，不读取 Cookie、密码或浏览器历史。章节内容只在本地浏览器中处理，用于当前创作者平台页面的章节填写。写入失败时，扩展会在本地保存当天错误日志；日志不会上传服务器，旧日志会在日期变化后自动清除。

使用前请确认：

- 已打开抖音或小红书网页端视频发布页。
- 已上传视频，并且页面显示章节编辑功能。
- 抖音会写入章节名和简介；小红书当前只写入章节名。

## 权限用途说明

- `activeTab`：用于和当前正在操作的创作者发布页通信。
- `scripting`：用于在旧标签页未自动加载扩展脚本时注入页面脚本。
- `storage`：用于在本地保存当天错误日志，方便用户主动复制给维护者排查写入失败问题。
- `https://creator.douyin.com/*`：用于在抖音创作者中心页面定位章节弹窗并辅助填写章节。
- `https://creator.xiaohongshu.com/*`：用于在小红书创作服务平台页面定位章节弹窗并辅助填写章节。

## 审核测试说明

1. 打开抖音创作者中心或小红书创作服务平台的视频发布页。
2. 上传一个超过 1 分钟的视频，使页面出现章节编辑入口。
3. 点击扩展图标。
4. 粘贴示例章节：

   ```text
   - 00:05-00:28｜阶段二总结：把规划基石阶段串成一张考试作战地图。
   - 00:28-00:49｜建体系到会规划：ISP 定方向，TOGAF 画蓝图。
   ```

5. 点击“写入抖音章节”或“写入小红书章节”。
6. 观察页面右下角进度面板，以及章节弹窗中新增的章节。
7. 扩展不会点击页面最终“发布”按钮。
