# 创作者章节助手

这是一个本地 Chrome 插件，用来把粘贴的章节文本批量写入创作者平台的视频章节编辑器。当前支持抖音创作者中心和小红书创作服务平台。

## 当前能力

- 支持抖音创作者中心视频发布页。
- 支持小红书创作服务平台视频发布页。
- 支持粘贴一行一条章节的文本。
- 支持 `00:05-00:28｜标题：简介` 格式。
- 写入前预览解析结果。
- 按当前平台校验章节限制：抖音章节名 12 字、简介 100 字；小红书章节名 11 字、不写入简介。
- 可选择写入前清空已有章节。
- 可选择自动截断超长标题和简介。
- 写入失败时可复制当天错误日志，便于排查平台页面变化。
- 不会点击最终“发布”按钮。

## 本地加载方式

1. 打开 Chrome。
2. 进入 `chrome://extensions/`。
3. 打开右上角“开发者模式”。
4. 点击“加载已解压的扩展程序”。
5. 选择本项目目录：

   `/Users/sunjianyin/Documents/wwwroot/auto-add-chapter`

## 使用流程

1. 在抖音或小红书的创作者视频发布页上传视频。
2. 确认页面已出现“视频章节”或“添加章节”区域。
3. 点击浏览器工具栏里的“创作者章节助手”。
4. 粘贴章节文本，例如：

   ```text
   - 00:05-00:28｜阶段二总结：把规划基石阶段串成一张考试作战地图。
   - 00:28-00:49｜阶段二从建体系到会规划：ISP 定方向，TOGAF 画蓝图，云网数安负责落地。
   ```

5. 检查预览和错误提示。
6. 按需要勾选“清空已有章节”“章节名超长时截断”“简介超长时截断”。
7. 点击“写入抖音章节”或“写入小红书章节”。
8. 页面右下角会显示执行进度。
9. 插件只保存章节弹窗，不会发布作品；最终发布仍需手动完成。

## 支持的章节格式

时间格式：

- `00:05-00:28`
- `00:05 - 00:28`
- `1:05-1:28`
- `01:02:03-01:03:20`

分隔符：

- `|`
- `｜`
- 空格

标题和简介使用第一个 `:` 或 `：` 切分。小红书当前只写入章节名，简介会被忽略；当标题为空时，会使用简介前 11 个字作为小红书章节名。

## 已知限制

- 抖音页面结构变化后，content script 可能需要更新。
- 小红书已有章节的删除控件不稳定；如果检测到已有章节但找不到删除按钮，插件会提示先手动清空后再重试。
- 插件不会绕过平台限制、账号权限或页面校验。

## 故障处理

如果提示 `Could not establish connection. Receiving end does not exist.`：

1. 先到 `chrome://extensions/` 找到“创作者章节助手”。
2. 点击“重新加载”。
3. 回到创作者发布页，刷新页面后再试。

新版插件已支持在旧页面没有 content script 时自动注入并重试；如果你刚更新过插件文件，仍然需要先在扩展管理页重新加载一次，让 Chrome 读取新的 `manifest.json` 权限。

如果抖音只能写入第一条章节，随后提示找不到“输入章节名”：

1. 先到 `chrome://extensions/` 重新加载插件。
2. 回到抖音发布页刷新页面。
3. 再重新执行导入。

插件会在每一条抖音章节写入前重新打开“手动添加”输入区，适配抖音在点击“确定”后收起输入框的页面行为。

如果小红书写入失败：

1. 点击插件弹窗里的“复制错误日志”。
2. 把复制出的当天日志发给维护者。
3. 日志只保存在本地，并会在日期变化后自动清除旧日志。
